Довідка: Профіль

-----Вміст довідки ----------

ОПИС

Модуль «Профіль» призначений для керування настроюваними полями профілю користувача.

ВСТАНОВИТИ/ВИДАЛИТИ

Ніяких особливих заходів не потрібно, дотримуйтесь стандартного процесу встановлення – розпакуйте папку /profile у каталог ../modules. Встановіть модуль через Адмін -> Системний модуль -> Модулі.

Докладні інструкції щодо встановлення модулів доступні в Керівництво з експлуатації XOOPS

ІНСТРУКЦІЯ З ЕКСПЛУАТАЦІЇ

Щоб налаштувати цей модуль, потрібно:

  • Налаштуйте свої параметри для модуля (див. «Параметри») і додатково блок «Профіль», якщо ви збираєтеся ним користуватися (див. «Блоки»).
  • Відредагуйте існуючі категорії або додайте нові.
  • Відредагуйте існуючі поля або додайте нові. Тут ви можете вказати, які поля в якій категорії будуть видимі та чи будуть вони видимі під час реєстрації користувача.
  • Визначте порядок кроків реєстрації.
  • І, нарешті, ви можете встановити дозволи для окремих полів – які можна редагувати, а які – шукати.

Заходи захисту від спаму

Щоб мінімізувати спам-реєстрацію, виконайте такі дії:

  • перейдіть до модуля Protector в адміністраторі, перейдіть до Preferences, а потім внизу, у цій опції: «Зупинити спам на форумі» Перевіряє дані POST на наявність спамерів, зареєстрованих у базі даних www.stopforumspam.com.
    Установіть значення "Заборонити IP (без обмежень)"

  • в /class/captcha/config.php переконайтеся, що встановлено режим «текст»:

    return $config = array(
    'disabled' => false, // Вимкнути CAPTCHA
    'mode' => 'text', // режим за замовчуванням, ви можете вибрати 'text', 'image', 'recaptcha' (потрібен ключ API)
    'name' => 'xoopscaptcha', // назва captcha
    'skipmember' => true, // Skip CAPTCHA check for members
    'maxattempts' => 10, // Максимальна кількість спроб для кожного сеансу
    );

  • У модулі «Профіль» перейдіть до «Адміністратор» і на кроці «Основне» за адресою:
    /modules/profile/admin/step.php?id=1
    встановіть "Зберегти після зупинки" на "Ні"


  • У налаштуваннях профілю встановіть для параметра «Використовувати Captcha після другого кроку реєстрації» значення «Так» (за умовчанням)
