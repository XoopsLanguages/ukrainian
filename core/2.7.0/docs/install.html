Встановлення XOOPS

Перше встановлення

Передмова:

XOOPS є відкритим кодом Об'єктно-орієнтована система веб-видання, написана на PHP. Це ідеальний засіб для розробка малих і великих динамічних веб-сайтів спільнот, внутрішньокорпоративних порталів, корпоративних порталів, веб-журналів і багато іншого.

XOOPS випущено згідно з умовами GNU General Public License (GPL) версії 2 або новішої, яку можна вільно використовувати та змінювати. Поширювати його можна безкоштовно, доки ви дотримуєтеся умов розповсюдження GPL.

Вимоги

  • WWW сервер (Apache, NGINX, IIS тощо)
  • PHP 8.2 або вище, рекомендовано 8.4+
  • MySQL 5.7 .8 або вище, рекомендується MySQL 8.4.5+

Перед встановленням

  1. Правильно налаштуйте сервер WWW, PHP і сервер бази даних.
  2. Підготуйте базу даних для свого сайту XOOPS.
  3. Підготувати обліковий запис користувача та надати йому доступ до бази даних.
  4. Зробити доступними для запису каталоги та файли: %s
  5. З міркувань безпеки настійно радимо перемістити два наведені нижче каталоги з корінь документа і змініть назви папок: %s
  6. Створіть (якщо ще немає) і зробіть каталоги доступними для запису: %s
  7. Увімкніть cookie та JavaScript у вашому браузері.

Встановлення локально

Якщо ви використовуєте локальне середовище для розробки або тестування, переконайтеся, що у вас є попереднє вимоги виконані. Коли це буде зроблено, скопіюйте вміст htdocs (з XOOPS файл розповсюдження) до кореневого шляху документа вашого веб-середовища. Після того, як файли будуть скопійовані туди, ви можете почати встановлення, вказавши у своєму браузері "http://yoursite.com". Це почне процес встановлення.

Встановлення на розміщеній платформі

Якщо ви працюєте в розміщеному середовищі, розпакуйте файли XOOPS локально або на сервері, якщо у вас є доступ через оболонку (тобто SSH). Після цього обов’язково перемістіть або скопіюйте всі файли XOOPS із htdocs у ваш кореневий веб-каталог (ваш постачальник послуг зазвичай надає це місце з маршрутами). Після того, як файли будуть скопійовані туди, ви можете розпочати встановлення, вказавши свій браузер "http://yoursite.com". Це почне процес встановлення.
